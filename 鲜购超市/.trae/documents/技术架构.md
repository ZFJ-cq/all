## 1. 架构设计

```mermaid
flowchart TD
    "A[React 前端应用]" --> "B[Zustand 状态管理]"
    "B" --> "C[localStorage 持久化]"
    "A" --> "D[商品数据 JSON]"
    "A" --> "E[搜索过滤模块]"
    "A" --> "F[购物车模块]"
    "F" --> "B"
```

纯前端架构，无需后端服务。所有商品数据以 JSON 格式内嵌于前端代码中，购物车状态通过 Zustand 管理，并同步至 localStorage 实现持久化。

## 2. 技术说明

- 前端：React@18 + TypeScript + Tailwind CSS@3 + Vite
- 初始化工具：vite-init（react-ts 模板）
- 状态管理：Zustand（含 persist 中间件实现 localStorage 持久化）
- 后端：无
- 数据库：无（使用内嵌 JSON 模拟数据）

## 5. 数据模型

### 5.1 商品分类数据

```typescript
interface Category {
  id: string;
  name: string;
  icon: string;
}
```

### 5.2 商品数据

```typescript
interface Product {
  id: string;
  name: string;
  description: string;
  price: number;
  image: string;
  categoryId: string;
  unit: string;
}
```

### 5.3 购物车条目数据

```typescript
interface CartItem {
  productId: string;
  name: string;
  price: number;
  image: string;
  quantity: number;
  unit: string;
}
```

### 5.4 Zustand Store 定义

```typescript
interface CartStore {
  items: CartItem[];
  addItem: (product: Product) => void;
  removeItem: (productId: string) => void;
  updateQuantity: (productId: string, quantity: number) => void;
  clearCart: () => void;
  getTotalPrice: () => number;
  getTotalCount: () => number;
}
```
